About This Site

Content

This site has 10 different pages all informing the visitor about different aspects of snowbirds. Most pictures were taken by the site creator and any pictures that were not are referenced. The reference link is located at the bottom of each page with external images.

Design

This site uses a modern and clean design. The design was inspired both by the iOS 11 user interface guidelines and Snowbird's actual website, snowbird.com. There are two CSS files, one for the navigation bar and another for everything else. There are very few inline styles. These are used because there are a few elements that need a minor change from the larger class they are part of. It is more efficient to add one inline style change rather than make a whole new class. All the pictures in the site show the natural beauty of the resort and make it a desireable place to visit.

Graphics and Layout

This site is very repetitive. Each page follows the same style with a related header image and easy to read titles and descriptions. The global navigation is fixed and available everywhere on the site. Page titles are accurite and the features of the site are compatable with all web browsers.

Content in Depth

This site includes a trail map of Snowbird on the Skiing page. All photo references are cited on this page.

Code

SSI was used for the navigation bar of the site. The code is organized based and comments make it easy to read.
